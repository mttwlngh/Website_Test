# Website Test

Kleine Test-Website mit einem zentrierten "Hallo".

## Lokal testen

- `npm run dev` startet einen lokalen Server auf http://localhost:5173
- `npm test` prüft, ob der Text "Hallo" vorhanden ist

## Deployment (Vercel)

- Repository zu GitHub pushen
- In Vercel "New Project" wählen und das Repo verbinden
- Framework: "Other" (statische Seite)
- Build Command leer lassen, Output Directory leer lassen
